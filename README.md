# Integração com API de terceiro- Consumo de serviços da Brasil API

Este repositório se trata da reprodução prática de projeto de integração com a Brail API fornececida pelo canal Artigo Tech - vídeo 'Como consumir API de Terceiro (Brasil API), com C# e .NET 6 | C# Na Prática - EP. 7 (parte 1)'.

O intuito do projeto é consumir serviços de bancos e ceps onde é possível buscar por uma lista de bancos, buscar um banco pelo através de código identificador e buscar por um endereço através de seu cep.
